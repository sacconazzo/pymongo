# pymongo
prove restAPI mongodb e flask

- GET '/employees' - list
- GET '/employees/<string_find>' - filter by name
- GET '/employee/<employee_name>' - insert new name
- POST '/employee' - insert new object with free properties
- DELETE '/employee/<oid>' - delete an object
